Bash

The following script takes three parameters (year, model, and new or used) and then accesses a mysql database to determine the car sales, dealer cost for all cars sold, total amount paid by buyer for all the cars sold, and total list price for all the cars sold. After determining the values the information is returned onscreen. This script shows my ability to use Bash to interact with a mySQL database.
